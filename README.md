# FlyDreamAir

FlyDreamAir is a multi-page airline booking website built with HTML, CSS and JavaScript.

## Pages

- Home
- Book a Flight
- Manage Booking
- Flight Status
- Contact

## Main Features

- Flight search by route and date
- Flight selection
- Passenger details form
- Seat selection
- Add-on travel services
- Booking review and confirmation
- Booking history using localStorage
- Booking cancellation
- Flight status lookup
- Responsive layout

## Technologies

- HTML
- CSS
- JavaScript
- localStorage
- GitHub Pages
